# 프로젝트의 계기
1. 뉴스를 보다 우연히 코스피 지수가 9000을 돌파한 것을 보았고, 주식을 검색해보니 삼성전자가 30만 원을 돌파했다는 사실을 알아챘음.
2. 예전에 8만 원에 샀다가 코로나 시즌을 지나고 81000원에 팔았던 기억이 생각나서 주식 검색 프로그램을 만들어보고 싶었음.
3. 그리하여 이 프로그램을 제작하게 되었음.

# 자기평가

## 잘했던 점:
- AI를 활용하여 소스코드 개발부터 파일 나누기, 패키지화, 그리고 테스트용 파일 작성과 깃허브에 커밋하는 것까지 이어서 실무적용능력을 보여주었다.
- 원하는 기능을 실시간으로 적용해 추가하고, 개량하였다.
- 테스트까지 전부 통과할 수 있었다.

## 아쉬웠던 점: 
- AI 사용량이 다 되면 작업을 못 한다는 점, 그리고 만들어둔 저장소가 아닌 다른 저장소를 VScode 내부에서 만드는 편이 더 편해서 못 쓰고 폐기처분해야했던 점이 아쉬웠다.
- customtkinter로 UI를 만들 때 tight_layout이랑 안 맞아서 UI 디자인을 살리면서 실행 시 사소한 에러가 대량으로 나오는 걸 방지할 수 없어 아쉬웠다. 
- 주식 데이터를 불러오는 yfinance의 한계로 특정 시간단위에서 중간중간 데이터가 누락되는 현상을 해결하지 못한 점이 아쉬웠다.

## 개선하고 싶은 점
- 뉴스 기사를 조금 더 상세히 불러오는 방법을 만들어보고 싶다.
- 매수/매도 기능을 추가해 실제 주식 앱처럼 사용해보고 싶다.(yfinance를 불러오는 느린 방식이 아니라 실제 금융 앱이 쓰는 것처럼 빠르고 정확한 방식을 사용해보고 싶다)
