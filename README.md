# 📈 주식 시장 대시보드 (Stock Dashboard)

## 1. 프로젝트 개요
**주식 시장 대시보드**는 사용자 지정 관심 종목의 실시간 가격, 재무 기본 지표(Fundamentals), 주가 차트, 그리고 최신 뉴스를 한눈에 파악할 수 있는 파이썬 기반의 GUI 애플리케이션입니다. 
객체지향 프로그래밍(OOP) 원칙을 적용하여 데이터 수집 모듈과 GUI 모듈을 분리하였으며, 패키지 형태로 배포 가능하도록 설계되었습니다.

## 2. 설치 방법
본 패키지는 Python 3.8 이상 환경에서 작동합니다. 가상환경을 생성한 후 설치하는 것을 권장합니다.

```bash
# 1. 가상환경 생성 및 활성화 (Windows 기준)
python -m venv .venv
.venv\Scripts\activate

# 2. 패키지 다운로드 및 디렉터리로 이동
cd stonks

# 3. 필수 라이브러리 및 패키지 설치
pip install .

3. 빠른 시작 (Quick Start)

패키지 설치가 완료되면, 터미널에 아래 명령어를 입력하여 대시보드를 즉시 실행할 수 있습니다.

가장 확실한 실행 방법 (추천):

python -m stock_dashboard.gui

단축 명령어로 실행하기 (Scripts 환경변수 PATH 설정이 된 경우):

run-dashboard

4. 주요 기능 설명

  - 관심 종목 관리 (Watchlist): 스마트 검색기능(한/영 지원)을 통해 관심 종목을 추가하고 실시간 변동률을 확인합니다.
  - 인터랙티브 차트: 1시간부터 5년까지의 기간 설정이 가능하며, 라인 차트 및 캔들스틱 차트를 지원합니다. (거래량 및 SMA 보조지표
    포함)
  - 기업 기본 지표: PER, EPS, 매출, 시가총액 등 기업의 핵심 재무 지표를 제공합니다.
  - 실시간 뉴스 연동: 선택한 종목과 관련된 최신 글로벌/로컬 뉴스를 실시간으로 수집하여 보여줍니다.
  - 데이터 업데이트 설정: 사용자 편의에 맞게 데이터 자동 갱신 주기(10초~5분)를 설정하거나 수동으로 새로고침할 수 있습니다.

5. 테스트 실행 방법

본 프로젝트는 pytest를 활용한 자동화된 단위 테스트와 pycodestyle을 통한 PEP 8 코드 스타일 검증을 포함하고
있습니다.

# 1. 단위 테스트 실행 (정상 및 엣지 케이스 포함)
python -m pytest

# 2. 코드 스타일 검사 (black 포매팅 적용 완료)
python -m pycodestyle stock_dashboard tests
